# and-or-graph-search

This is project resolving n-queens problem using And-Or graph search algorithm

# Running

$ pip install tkinter

$ pip install pilow

$ app.py

- Step 1: Enter a number of queens and number of interactions

- Step 2: Click Simulated Annealing button to find solution
